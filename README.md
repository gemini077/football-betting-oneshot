# Football Betting OneShot

Football Betting OneShot 是一个以概率、盘口、价格纪律和严格赛后结算为核心的足球比赛分析工作台。

## 在线工作台

GitHub Pages 部署完成后，入口为：

`https://gemini077.github.io/football-betting-oneshot/`

## 自动化范围

- 每天按北京时间更新体彩赛程并重建统一主页。
- 自动维护赛后复盘队列、复盘总表和静态网页。
- 每次发布前运行完整测试。
- 不自动分析体彩开售的全部比赛；赛前分析只处理用户选择的场次。
- 未经明确“锁单/已下单”，任何候选方案都保持未锁单。

## 安全边界

公开仓库不包含渠道账号、密码、Cookie、浏览器会话、实时抓取原始包或本地扩展配置。实时滚球赔率桥接仍在用户本机运行。

详细模型与数据约束见项目根目录的编号文档。
